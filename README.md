# 🧪 QA Automation Framework – OrangeHRM (Practice Project)

This repository contains a complete **UI automation framework** designed to test the OrangeHRM demo application:  
🔗 https://opensource-demo.orangehrmlive.com/

The project demonstrates strong knowledge in:

- C# + SpecFlow (BDD)
- Selenium WebDriver
- ReqRoll BDD style
- Page Object Model (POM)
- Hooks & lifecycle management
- Web tables, alerts, notifications, checkboxes, and popups
- Clean, readable, natural-language test cases
- Extent Reports, Screenshots and Serilog

---

## 🚀 Tech Stack

| Component | Description |
|----------|-------------|
| **Language** | C# |
| **BDD Framework** | SpecFlow (Gherkin) |
| **UI Automation** | Selenium WebDriver |
| **Design Pattern** | Page Object Model |
| **Runner** | NUnit / xUnit |
| **Reporting** | SpecFlow built-in |
| **Drivers** | Local Chrome/Edge WebDriver |
